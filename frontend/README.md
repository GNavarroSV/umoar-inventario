# My Fullstack App - Frontend

This is the frontend part of the My Fullstack App, built using Next.js. This application serves as the user interface and communicates with the backend API.

## Getting Started

To get started with the frontend, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd my-fullstack-app/frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Navigate to `http://localhost:3000` to see the application in action.

## Project Structure

- `app/`: Contains the main application pages and layout.
- `components/`: Contains reusable React components.
- `lib/`: Contains utility functions, including API calls to the backend.
- `next.config.js`: Configuration file for Next.js.
- `package.json`: Lists dependencies and scripts for the frontend.
- `tsconfig.json`: TypeScript configuration for the frontend.
